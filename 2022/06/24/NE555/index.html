NE555内部原理

NE555是一个非常重要的集成芯片，它价格低廉，能实现的功能非常多。下面来看一下NE555的内部等效电路图:

该芯片由三个分压电阻、两个比较器、一个RS触发器、一个非门组成。

首先看分压电阻，由于比较器输入端虚断，所以右侧电路不对其造成影响，三个电阻串联分压。

然后再看两个比较器。比较器的工作原理是如果同相输入端（+）的电位大于反向输入端，则输出高电平，反之输出低电平。由于放大器的增益A非常大，因此两个输入端电位差别再小也能检测出来。

最后看一下触发器。

S是设置端（set），R是复位端（reset)。在555电路中，只有Q非发挥作用，Q不发挥作用。

实际上这个触发器有另一个触发引脚，即PIN4 R’。它的定义是主复位（main reset）。当R’接低电平时，它上面的三极管会被导通。根据引脚图：

当PIN4处于低电平时，555会复位，因此R’的优先级是最高的，只有PIN4为高电平的时候555才能正常工作。

1 1 0/1 0
1 0 1 1
1 0 0 保持不变

那么根据上述内容，我们可以得出555所有工作状态的表格

PIN4 PIN2 S PIN6 R PIN3
1 >1/3 Vcc 0 >2/3 Vcc 1 0
1 >1/3 Vcc 0 <1/3 Vcc 0 保持之前的状态
1 <1/3 Vcc 1 无意义 无意义 1
0 无意义 无意义 无意义 无意义 0

NE555的典型应用

方波发生器

刚上电时，PIN4为1，使得555开始工作。PIN2和PIN6短接了，由于刚上电时C1相当于短路，所以PIN2<1/3 Vcc，PIN3输出高电平。

等到电容充满电后PIN2>1/3 Vcc，PIN6>2/3 Vcc，因此PIN3输出低电平。

而PIN3前有一个非门，当PIN3为低电平时，PIN7三极管的基极为高电平，三极管导通。此时我们再来看一下在这个电路中555芯片内部的电路图：

基极的高电平会导致三级管导通，电容中的电将会被放掉，从而回到一开始的状态。如此往复就能形成方波。

因此这里电容能够放电是形成方波的必要条件。而我一开始搭建时，电位器短路了，导致PIN7的电压始终为5V，而基极的电压最大也只有5V，因此三极管反向截止，电容不能正常放电，从而PIN3始终输出低电平。

经过一番排查后，将电位器换为了电阻，电路正常工作，只不过占空比不对。但我们知道，只要选择了恰当的电阻，就能使占空比达到50%，形成标准的方波。

占空比公式为：
$$
(R1+R2)/
(R1+2R2)
$$
